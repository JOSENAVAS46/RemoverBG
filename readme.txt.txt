Tener en cuenta que debes de tener Instalado Python en tu computador, junto a pip.

	1. Ingresar en https://www.remove.bg/
	2. Crear una cuenta
	3. Ir a https://www.remove.bg/api
	4. Click en Get API Key
	5. Click en New API Key
	6. Colocarle un Nombre a la Api Key
	7. Copiar la API Key
	8. Instalar dependencias:

		Ejecuta el siguiente comando para instalar las dependencias desde el archivo requirements.txt:
		
			pip install -r requirements.txt

	9. Ejecutar el programa directamente:
			Ejecuta tu programa directamente usando Python:
				
				python removerBG.py
